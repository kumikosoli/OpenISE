# 数据结构与算法-交通

## 课程信息
- 授课教师：王军波，蔡铭
- 学分：3学分
- 考核方式：笔试 + 机试

## 复习策略
- 课程重要程度非常高，课程知识会贯穿（cs相关）考研 / 保研 / 求职，认真学不亏。
- 机试难度中等，（交通目前是）代码填空。笔者采用硬背代码（`数据结构与算法 代码合集/数据结构下半学期代码合集.docx` ，之所以分出下半学期代码合集是课上有讲机试以下半学期内容为主）的方式复习，效果不错。
- 笔试难度较高，会有手写代码，整体分都不高，复习成本比较高。
- 附带 `数据结构重点整理.pdf` ，根据课件所标注的重点内容制作，可用性中等，实际考试内容超过该文件。**复习内容可能有改动，此文件仅供参考。**
-  笔者为复习该课程在网络上搜集了相关资料放在 `课外` 文件夹内，同学们按需使用即可。课程所用参考书（清华紫书）有配套题集，实用性未知。
-  再次回应开头，未来有规划cs相关经历的同学推荐认真学习，要是能养成刷题（力扣/洛谷etc）习惯更好，未来有用的。
